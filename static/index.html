Deno Oak Template!

By @RoBlockHead

This Template is a basic template that shows what you can do with Deno and the Oak web Framework!
  • This file is served using the static file server within mod.ts
  • All non-static paths are served through routes.ts
  • You can see an example of routing by going to /hello.
  • There's also an example of redirecting at /redirect
  • You can use parameters and queries at /demo.
  • Login Demo at /login.
